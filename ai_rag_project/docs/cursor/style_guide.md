# Style Guide - Chef-AI Frontend

## Primary Colors
• **Chef-Red**: #E63946 (primary action buttons, accents)
• **Kitchen-White**: #FFFFFF (backgrounds, clean space)
• **Shadow-Gray**: #2A2A2A (text, serious content)
• **Light-Gray**: #F8F9FA (subtle backgrounds, dividers)

## Typography System
• **Headings**: 'Inter' font, 600 weight, tight letter-spacing
• **Body Text**: 'Inter' font, 400 weight, 1.5 line-height
• **Recipe Titles**: 'Inter' font, 700 weight, 2.5rem size
• **Ingredient Lists**: 'Inter' font, 400 weight, 1.2rem size

## Readability Rules
• **Contrast Ratio**: Minimum 4.5:1 for WCAG AA compliance
• **Font Sizes**: Minimum 16px for body text, scalable
• **Line Height**: 1.5 for body text, 1.2 for headings
• **Character Limit**: 65-75 characters per line maximum

## Kitchen Environment Optimization
• **Large Touch Targets**: Minimum 44px for finger interaction
• **High Contrast**: Enhanced contrast for bright kitchen lighting
• **Bold Typography**: Heavier font weights for glare resistance
• **Simple Icons**: Clear, universally understood symbols

## Responsive Typography
• **Mobile**: 16px base, 1.375 scale ratio
• **Tablet**: 18px base, 1.25 scale ratio  
• **Desktop**: 20px base, 1.25 scale ratio
• **Large Screens**: 22px base, 1.125 scale ratio

## Component Styling
• **Buttons**: Chef-Red background, white text, rounded corners
• **Cards**: White background, subtle shadows, clean borders
• **Inputs**: Light borders, focus states with Chef-Red
• **Navigation**: Dark background, high contrast text
